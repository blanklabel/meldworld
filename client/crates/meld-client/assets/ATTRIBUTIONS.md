# Asset attributions

## `monsters/`, `foliage/` (tree/statue/decor), `ground/`, `resources/`, `fx/`

Pixel-art tiles from **Dungeon Crawl Stone Soup** (the `rltiles` set), which is
**public domain**. Per the tileset's own license
(`crawl-ref/source/rltiles/license.txt`): the graphic tiles derive from the
public-domain roguelike tileset **RLTiles** (http://rltiles.sf.net); DCSS's own
original tiles are released **CC0** (`crawl-ref/docs/license/cc0.txt`). No
attribution is required for CC0 / public-domain works — this file is a courtesy
record of provenance.

- Source repo: https://github.com/crawl/crawl (`crawl-ref/source/rltiles/`)
- License: Public Domain (RLTiles) / CC0 (DCSS-original tiles)
- Downloaded: 2026-07-18

Sub-sources within that set:
- `monsters/*` — `rltiles/mon/{animals,amorphous,humanoids,demihumanoids,undead,dragons}/`
- `foliage/*` — `rltiles/mon/fungi_plants/`, `rltiles/dngn/{trees,statues,decor}/`
- `ground/*` — `rltiles/dngn/floor/` (grass/dirt/sand/moss)
- `resources/*` — `rltiles/item/{misc,gem}/`
- `fx/portal_*` — `rltiles/dngn/gateways/`
